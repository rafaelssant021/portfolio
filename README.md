# Portfólio - Rafael Santos

Acesse o projeto: https://rafaelssant021.github.io/portfolio/

##  Sobre
Este é o meu portfólio pessoal desenvolvido para apresentar meus projetos, habilidades e evolução como desenvolvedor.
O objetivo deste site é reunir em um só lugar os principais trabalhos que venho desenvolvendo durante meus estudos em Análise e Desenvolvimento de Sistemas.

## Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript

## Funcionalidades
- Apresentação pessoal
- Exibição de projetos
- Seção de habilidades
- Layout responsivo

## Aprendizados
Durante o desenvolvimento deste portfólio, pratiquei:
- Estruturação de páginas com HTML
- Estilização com CSS
- Interatividade com JavaScript
- Organização de projetos para apresentação profissional
